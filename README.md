# arduino-starter

## BEFORE START

```
cd /opt/
sudo wget https://downloads.arduino.cc/arduino-1.8.15-linux64.tar.xz
sudo tar -xf arduino-1.8.15-linux64.tar.xz
sudo rm -f arduino-1.8.15-linux64.tar.xz
sudo chmod -R 777 arduino-1.8.15
cd arduino-1.8.15
wget https://downloads.arduino.cc/arduino-cli/arduino-cli_latest_Linux_64bit.tar.gz
tar -zxf ./arduino-cli_latest_Linux_64bit.tar.gz
```
